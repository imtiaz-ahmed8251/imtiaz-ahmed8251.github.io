Automatidata Exploratory Data Analysis

Project Overview

The NYC Taxi & Limousine Commission has contracted with Automatidata to build a regression model that predicts taxi cab ride fares. In this part of the project, the data needs to be analyzed, explored, cleaned and structured prior to any modeling.

Project Objective

The purpose of this project is to conduct exploratory data analysis on a provided data set with the aim of learning more about the variables.

Stages of EDA Process

  • Import, Link and Load Data
  • Data Exploration
  • Data Cleaning
  • Creating Visualizations
  • Evaluate and Share Results

Outcome

As a result of the conducted exploratory data analysis, the Automatidata data team considered trip distance and total amount as key variables to depict a taxi cab ride.
