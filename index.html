ABOUT ME

Imtiaz is motivated by the belief that data can be used to make better decisions. He wants to be able to make it easier for people to understand code and use data efficiently. Imtiaz believes in open data and open source software. Imtiaz primarily uses Python, but also has experience with R and Scala.

Imtiaz has worked at leading cyber security companies like Palo Alto Networks, Cloud Raxak and McAfee on network security, endpoint security and configuration and compliance. Imtiaz graduated from the School of Information at UC Berkeley where he explored various facets of data science, data visualization and user research.
